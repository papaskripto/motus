## motus

#### motus is a financial expense tracker. It works just like a notes app (both offline and online) where users can:

- Create accounts
- Delete their accounts
- Add expense(s)
- Delete expense(s)
- See their total expenses per month

#### Tech Stack
- HTML
- CSS
- JavaScript
- IndexedDB
